Quick Setup

The original manual makes an important point: this is a third-party camera module, not the official Raspberry Pi camera, so system settings must be changed before the camera will work.

Important: This camera is not plug-and-play. You must modify the system configuration before use. Otherwise, the camera may not light up.
System reminder: The original manual recommends using the latest Raspberry Pi system version. It specifically mentions a system reference around 2025-10-01 / Kernel 6.12.
1

Connect the Camera

Install the camera module on your Raspberry Pi and boot into the desktop or terminal environment.

2

Modify config.txt

Open Terminal and enter the setup command exactly as shown below.

3

Restart and Test

Reboot the system and run the preview command to confirm the camera is working.

sudo echo -e "camera_auto_detect=0 dtoverlay=imx708,cam0 dtoverlay=imx708,cam1" | sudo tee -a /boot/firmware/config.txt
sudo reboot
sudo rpicam-hello
Success check: You should see a camera preview window for about five seconds.

Preview and Photo Capture

The original manual explains how to open the camera preview window and how to capture JPEG photos.

Open the Preview Window

Preview the camera feed continuously:

sudo rpicam-hello -t 0

Preview in full screen:

sudo rpicam-hello -f -t 0

To close the preview window, press Alt + F4 or click the close button.

Capture a JPEG Photo

Capture a full-resolution JPEG image:

sudo rpicam-jpeg -o test.jpg

This command shows a short preview and then saves the image as test.jpg.

Capture at 1080p resolution:

sudo rpicam-jpeg -o test.jpg -t 2000 --width 1920 --height 1080

Capture at VGA resolution:

sudo rpicam-jpeg -o test.jpg -t 2000 --width 640 --height 480

Video Capture

For video recording, the manual uses rpicam-vid. The command opens a preview window and writes the video stream to an output file.

Record a 10-second Video

sudo rpicam-vid -t 10000 -o test.h264

This records a 10-second video and saves it as test.h264.

Record 1080p Video

sudo rpicam-vid --width 1920 --height 1080 -o full_hd.h264

Use this mode for Full HD recording.

Record 720p at 50fps

sudo rpicam-vid --width 1280 --height 720 --framerate 50 -o 50fps.h264

Use this mode for smoother motion and faster-moving subjects.

Good to know: The original manual notes that 1920 × 1080 should be treated as the largest standard video resolution unless otherwise stated.

Autofocus and HDR Imaging

Autofocus

When using the IMX708 camera module, autofocus is enabled automatically in continuous mode. The lens will move as needed to keep the center of the image in focus.

You can also disable autofocus and set the focus manually by using --lens-position. The value is measured in dioptres, which is the reciprocal of the focus distance.

  • Focus at about 0.1 m → use value 10
  • Focus at infinity → use value 0

Example:

sudo rpicam-vid --lens-position 0.5 -o 2_metres.h264

This sets focus to about two metres and keeps the lens fixed there.

High Dynamic Range (HDR)

HDR helps improve image quality in scenes with strong differences between bright and dark areas.

Example:

sudo rpicam-vid --hdr -o hdr.h264

The original manual explains that non-HDR capture can go up to 4608 × 2592, but HDR capture is limited to 2304 × 1296. H.264 video files are limited to 1920 × 1080.

Dual Camera Support on Raspberry Pi 5

Raspberry Pi 5 supports two cameras. When testing, you must specify which camera to use. Without this parameter, cam0 is selected by default.

sudo rpicam-hello -t 0 --camera 0 sudo rpicam-hello -t 0 --camera 1
Tip: To run two cameras, open two Terminal windows and run one camera command in each window.

Safety Instructions

The points below preserve the safety guidance from the original manual, rewritten in a clearer format.

Before You Connect

  • Shut down your Raspberry Pi completely before connecting the camera.
  • Disconnect the board from external power before installation.

Ribbon Cable Handling

  • If the cable comes loose, pull the locking tab forward first.
  • Insert the ribbon cable with the metal contacts facing the circuit board.
  • Push the locking tab back into place after inserting the cable.
  • Do not fold, twist, or strain the ribbon cable.

Environment and Storage

  • Operate the camera in a dry environment between 0–50°C.
  • Do not expose the module to water or moisture.
  • Do not place it on a conductive surface while in use.
  • Store the device in a cool, dry location.
Handle with care: Avoid mechanical or electrical damage to the printed circuit board, connectors, and ribbon cable.
